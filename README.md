# css-foundation
quick start CSS - css-foundation support to create stylesheet of your project

## Installation

```sh
cp dst/foundation.css [your project]
```

or

```sh
cp dst/foundation.min.css [your project]
```

or

```sh
cp -r sass/ [your project]
# and compile when use
```
